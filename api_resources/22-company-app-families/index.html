Company App Families

/companies/{companyName}/appfamilies/
An collection of Apps that belong to a Company
A collection of Apps belonging to a Company. This is often used to associate a 'sandbox' version of an app with a 'production' version of an app. Apigee does no special processing on such collections, except to maintain the relationship among apps that are contained in the family that is created using this method.
{
  "apps" : [ "weatherapp_sandbox", "weatherapp_prod" ],
  "name" : "WeatherApps"
}
