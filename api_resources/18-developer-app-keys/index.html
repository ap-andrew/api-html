Developer App Keys

/developers/{developerName}/apps/{appName}/keys/{consumerkey}
The Consumer Key (or "API Key") provisioned for an App
Crenetial pairs consisting of consumer key and consumer secret are provisioned by Apigee to apps for specific API products. APigee maintains the relationship between key and API product, enabling APi product to be added to and removed from consumer keys. A single key can be used to access multiple API products. Keys many be manually or automatically approved for API products--that setting is part of the API product configuration. A key must approved and and approved for an API product to be capable of accessing any of the URIs defined in the API product.
