API Products

/apiproducts
A bundle of API resources (URIs) plus rate limit settings and relevant metadata
Apigee provides API products to enable you to bundle and distribute your APIs to multiple developer constituencies simultaneously, with having to modify any code. As part of the app provisioning flow in Apigee Enterprise, developers are presented with one or more API products to select. API products provide the basis for access control in Apigee. The developer app is provisioned with a key and secret that enable the app to access the URIs bundled in the selected API product. The API supports multiple API products per app key.

List API Products Detail --> 
/apiproducts
Verb: GET
Description: Lists all API Products by name for an Organization
Auth:Basic Auth
Get count of keys/apps/developers/companies for API Product Detail --> 
/apiproducts/{apiproductName}?query=count&entity=keys/apps/developers/companies
Verb: GET
Description: Gets a count of one of the entities (either keys/apps/developers/companies ) associated with a given API Product
Auth:Basic Auth
Get list of keys/apps/developers/companies for API Product Detail --> 
/apiproducts/{apiproductName}?query=list&entity=keys/apps/developers/companies
Verb: GET
Description: Gets the List of keys/apps/developers for an API product of an organization
Auth:Basic Auth
