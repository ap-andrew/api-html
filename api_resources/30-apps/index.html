Apps

/apps
Applications regsitered with an API Provider's organization
Applications registered with an API Provider's organization. Apps are registered in order to obtain credentials that enable access to one or more API products (or set of URIs). Registered apps have an extensible profile of metadata. Apps can be associated with developers (Developer Apps) or with Companies (Company Apps).
