App Family

/developers/{developerEmail}/appfamilies
A collection of Apps for a Developer
A collection of apps belonging to one developer. An App Family can contain as many apps an necessary, and the grouping is entirely arbitrary. 
{
  "apps" : [ "weatherapp_sandbox", "weatherapp_prod" ],
  "name" : "WeatherApps"
}
Create an App Family Detail --> 
/developers/{developerEmail}/appfamilies
Verb: POST
Description: Create an App Family for Developers
Auth:Basic Auth
Get App Family Detail --> 
/developers/{developerEmail}/appfamilies/{appFamily}
Verb: GET
Description: Gets the Apps in an App Family for a Developer
Auth:Basic Auth
Update an App Family Detail --> 
/developers/{developerEmail}/appfamilies/{appfamilyName}
Verb: POST
Description: Updates an existing App Family
Auth:Basic Auth
Remove App from App Family Detail --> 
/developers/{developerEmail}/appfamilies/{appFamily}/app/{appName}
Verb: DELETE
Description: Removes an App from an App Family
Auth:Basic Auth
Delete App Family Detail --> 
/developers/{developeremail}/appfamilies/{appFamily}
Verb: DELETE
Description: Deletes an App Family and all Apps it contains
Auth:Basic Auth
List App Families Detail --> 
/developers/{developeremail}/appFamilies
Verb: GET
Description: List all App Families in an Organization
Auth:Basic Auth
List App Families (Expanded) Detail --> 
/developers/{developeremail}/appfamilies?expand=true
Verb: GET
Description: An expanded list of all App Families in an Organization, listing Apps in the collection
Auth:Basic Auth
