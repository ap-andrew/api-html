Import an API :: /apis?action=import&name={apiName}

API resource: APIs

Verb: POST

Doc: It's often beneficial to work do configuration of API proxies locally. This method enables developers to upload an API proxy as a ZIP file from the local machine to the targeted Apigee organization. Once imported, the API revision must be deployed before it can be access at runtime.

Description: Uploads an API proxy from a local machine to Apigee

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/test/apis?action=import&name=passthrough -T {filename}.zip -H "Content-Type: application/octet-stream" -X POST
