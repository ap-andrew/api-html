Create an API :: /apis

API resource: APIs

Verb: POST

Doc: This method creates the framework of an API proxy in Apigee, providing an alternative to creating APIs via the UI. The API will not be accessible at runtime until a proxy endpoint and target endpoint are defined. Usually, this request is followed up with an export API proxy method, which brings the API proxy bundle to the developer's local machine, where AP proxy can be configured and then imported back into Apigee for execution.

Description: Create an API definition and an API proxy configuration on Apigee Enterprise

cURL: 
curl -u myname:mypass -H "Content-Type: application/json" https://api.enterprise.apigee.com/v1/o/{myorg}/apis
