Get Statistics for an Environment :: /environments/{environmentName}/stats/?select={metrics}&timeRange={startDate~endDate}&timeUnit={unit}

API resource: Analytics

Verb: GET

Doc:

Description: Gets statistics for a specified time range for the environment specified. Available staistics are message count, error count, and API count.

cURL: 
curl -u myname:mypass 'https://api.enterprise.apigee.com/v1/o/{myorg}/environments/test/stats/?select=message_count&timeRange=7%2F1%2F2012%2000%3A00~8%2F1%2F2012%2000%3A00&timeUnit=hour
