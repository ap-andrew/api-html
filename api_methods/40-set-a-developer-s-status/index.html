Set a Developer's Status :: developers/{developerName}?action={newStatus}

API resource: Developers

Verb: POST

Doc: Use the POST method to set the default status of a developer. Status types are active, inactive, and login_lock. By default, the status of a developer is set to inactive. An Apigee user with admin credentials can change the developer's status to 'active' using this API call.. If a developer's status is inactive, the developer is not allowed to create apps, view or update their profile, or view statistics, and any apps created by the developer are deactivated. If a developer's status is login lock, they cannot access their account or change their apps; however, their apps remain active.

Description: Sets the status of a developer

cURL: 
curl -u {myname:mypass} "https://api.enterprise.apigee.com/v1/o/{myorg}/developers/{developerID}?action=inactive"
