Getting Developer for an App :: /developers?app={appName}

API resource: Developers

Verb: GET

Doc:

Description: Get the Developer associated with an App

cURL: 
curl -u myname:mypass "https://api.enterprise.apigee.com/v1/o/{myorg}/developers?app=TestApp"
Request Payload:
Request Payload Elements:
Name Description Default Required?
Response Payload: 
  {
  "apps" : [ "MyTestApp" ],
  "email" : "developer2@apigee.com",
  "firstName" : "Woodre",
  "last_name" : "Wilson",
  "userName" : "wilson",
  "organizationName" : "gsorg",
  "status" : "active",
  "attributes" : [ {
    "name" : "region",
    "value" : "north"
  } ],
  "createdAt" : 1342258630656,
  "createdBy" : "defaultUser",
  "lastModifiedAt" : 1342258630656,
  "lastModifiedBy" : "defaultUser"
}
Response Payload Elements:
Name Description
apps: The name of the App that the search was performed on Detail
email: Email address of the developer of this App Detail
firstName: First name of the developer of this App Detail
last_name: Last name of the developer of this App Detail
organizationName: The Organization to which the Developer of this App belongs Detail
userName: Username of the Developer of this App (not used by Apigee) Detail
status: The status of the Developer of this App (Active, Inactive, login_lock) Detail
attributes: Arbitrary name/value pairs that constitute a custom profile Detail
createdAt: The Unix time value for the creation date of this entity Detail
createdBy: The username for the Apigee account holder who created this Developer profile Detail
lastModifiedAt: The Unix time value for the most recent modification date of this entity Detail
lastModifiedBy: The username for the Apigee account holder who most recently modified this Developer profile Detail
Errors
Auth: Basic Auth

Content Types: --- - ""
